# Expense Tracker

A simple and intuitive expense tracking application built with vanilla HTML, CSS, and JavaScript to help you manage your finances.

!<img width="1237" height="678" alt="image" src="https://github.com/user-attachments/assets/3deb7608-8151-47f4-af8d-1af517923d9b" />


## Features

- **Add Expenses**: Quickly add expenses with category, amount, and date
- **Track Spending**: View all your expenses in an organized list
- **Calculate Totals**: Automatically calculates your total expenses
- **User Authentication**: Simple login system with personalized welcome message
- **Data Persistence**: Expenses stored in browser's local storage
- **Clean Interface**: Modern, user-friendly design
- **No Dependencies**: Pure vanilla JavaScript - no frameworks needed!

## Technologies Used

- **HTML5**: Structure and markup
- **CSS3**: Styling and layout
- **JavaScript**: Functionality and interactivity
- **Local Storage**: Data persistence (hardcoded)

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/expense-tracker.git
```

2. Navigate to the project directory
```bash
cd expense-tracker
```

3. Open `index.html` in your web browser
```bash
# Or simply double-click the index.html file
```

That's it! No build process or dependencies required.

## Usage

1. **Login**: Sign in with your credentials
2. **Add Expense**: 
   - Select a category from the dropdown
   - Enter the amount
   - Choose a date
   - Click "Add Expense"
3. **View Expenses**: All your expenses will be displayed in the list below
4. **Track Total**: The total amount is automatically calculated and displayed
5. **Logout**: Click the logout button when you're done
